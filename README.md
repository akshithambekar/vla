# VLA

Main code repo for building our VLA system.

`simplevla` is the blog/process repo where we document decisions and progress.

This repo is where we actually build, test, and iterate.

## What Goes Here

- Model code
- Training scripts
- Experiments

## Current Focus

We are currently building a neural network from scratch.

## Goal

Build a strong first-principles foundation before scaling to larger VLA
components.
